# The Swaa Boutique

A modern and responsive fashion boutique website built using **HTML, CSS, and JavaScript** in **Visual Studio Code**.

This project showcases a luxury-inspired design with responsive layouts, some effects, product cards, blog pages, and interactive navigation.

# Features

- Responsive Mobile Design
- Modern Fashion Boutique UI
- Responsive Navbar & Hamburger Menu
- Product Grid & Cards
- Blog Page with Read More Navigation
- Login Page UI
- About Us Page
- Footer Section
- Smooth Hover Effects & Transitions

---

- HTML5
- CSS3
- JavaScript
- Google Fonts

#  How to Run the Project

## 1. Clone the Repository

git clone https://github.com/your-username/the-swaa-boutique.git

## 2. Open in VS Code

Open the project folder using Visual Studio Code.

## 3. Run the Website

Open index.html in your browser.

You can also use the Live Server Extension in VS Code for better development experience.
